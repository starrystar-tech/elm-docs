### 启动
* 安装nodejs 安装18以上版本

* 克隆 https://github.com/starrystar-tech/elm-docs.git 项目，并 Star 关注下该项目。

* 在目录下执行如下命令，进行启动：

```shell
# 安装 pnpm，提升依赖的安装速度
npm config set registry https://registry.npmmirror.com
npm install -g pnpm
# 安装依赖
pnpm install

# 启动服务
pnpm run docs:dev
```
出现如下界面，说明启动成功
